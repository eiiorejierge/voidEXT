How it works

  1. Click the bookmarklet on any page — a beach popup appears.
  2. Sign up or log in (handled entirely in the popup).
  3. Hit Generate Links to pull a fresh rotating batch from the vault.
  4. If a link is dead/blocked, tap blocked on it — it's reported and pulled from everyone's rotation.

The full link list never leaves the server. The popup only ever sees a small rotating slice, and only after you log in.
