How it works

  1. Click the bookmarklet on any page — a popup opens.
  2. Sign up or log in (handled entirely in the popup).
  3. Hit Generate Links to pull your daily set from the vault (5/day).
  4. Your set is saved — logging back in loads the same links.
  5. If a link is dead, tap blocked — it's reported and pulled from everyone's rotation.

The full link list never leaves the server. The popup only ever sees your small slice, and only after you log in.
