Beach-themed login bookmarklet and browser extensions.

## Beach Club Login Bookmarklet

A self-contained browser bookmarklet that injects a popup overlay with curved corners and a cool beach-themed UI on any webpage.

### Installation

1. Create a new bookmark in your browser (e.g., Chrome, Edge, Firefox, Safari).
2. Edit the bookmark.

### Visual Features

- **Backdrop-Blur Overlay**: Blurs the host webpage to focus on the login card.
- **Glassmorphism**: Translucent container card with HSL-tailored colors.
- **Sun & Clouds**: Pulsing animated sun and slow-drifting clouds.
- **Crashing Waves**: Parallax scrolling waves that ease to a sandy shore.
- **Isolated Iframe**: Uses an iframe container to prevent the host site's CSS styles from contaminating the beach theme.
- **Dismissible**: Press `Escape` (on parent page or inside iframe), click the close button `✕`, or click anywhere on the background overlay to dismiss the popup.
